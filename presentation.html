Navigate Kathmandu Valley
Public Transit

Sawari (सवारी) — meaning "ride" in Nepali — is a smart transit navigator with live tracking, AI-powered routing, fare estimates, and community-driven data.

AI Hackathon • Team Spark

 The Problem

Getting around Kathmandu by bus is confusing. There are dozens of routes, no official app, and unless you already know which bus to take, you're stuck asking strangers or taking a taxi.

 Key Features

AI-Powered Navigation

Natural language input — "Bagbazar to Basundhara". Groq LLM extracts locations, finds the best bus route including transfers.

Fare Calculator

Nepal DoTM tariff rates, bus vs microbus ranges, student/elderly discounts, realistic Rs 5 increments.

Live Vehicle Tracking

Real-time positions polled every 3 seconds with smooth animated movement and ETA estimates.

Carbon Savings

CO₂ comparison — car (170 g/km) vs bus (50 g/km). Shows grams or kilograms saved per trip.

GPS Integration

Real-time tracking, compass heading, accuracy circle, nearby stops discovery, one-tap "Use as Start".

Obstruction Awareness

Backend avoids active road blockages and protests. OSRM alternatives scored against obstructions.

Rating System

Rate routes and vehicles 1–5 stars. Running averages maintained server-side.

AI Chatbot

Groq-powered Llama 3.3 70B chatbot answers transit questions directly on the landing page.

Community Suggestions

Submit route corrections or missing stops. AI extracts tasks for one-click admin approval.

 Public Navigator

  • Set start/end by clicking the map, typing, or right-click menu
  • Natural language — "take me from X to Y" (Groq AI)
  • Direct and transfer route finding
  • OSRM road-snapped polylines with direction arrows
  • Draggable start/end markers & swap button
  • Multi-candidate stop search (800m radius)
  • Walking fallback as last resort
  • Route focus mode & recenter/fit button
  • Local + online autocomplete with LRU cache
  • Dark / Light theme with matched tile layers
  • Explore all routes — filter, click to highlight
  • Keyboard shortcuts: / ? Enter Esc T E G N
  • Responsive layout — mobile, tablet, desktop
  • Toast notifications & skeleton loading states
  • Stats bar: live clock, stop count, route count, vehicles
  • Share / copy route summary button

 Admin Dashboard

Workspace Layout

Command bar, layer panel, map canvas, inspector, status strip. Responsive for tablet & mobile.

Stops Management

Create, edit, delete stops with FontAwesome or custom image icons, color picker, and dependency checks.

Route Builder

Multi-step builder: click stops on map, drag-reorder, snap to road, preview polyline, color & style controls.

Vehicle Management

Assign to routes, toggle moving state, set bearing, upload images. Live markers on the map.

AI Assistant

Ctrl+I — natural language entity CRUD. "Create a stop called Balaju at 27.72, 85.30". Batch actions supported.

Undo / Redo

Command pattern for all mutations. Ctrl+Z undo, Ctrl+Y redo. Full action history in status bar.

 Tech Stack

Vanilla JavaScript

ES2020+, no frameworks, no bundlers. IIFE module pattern with event-driven store.

PHP 8+

LAMP stack on XAMPP. Session auth, flat JSON storage, file locking for concurrency.

Leaflet 1.9.4

Interactive maps with CARTO basemaps, custom markers, polylines, and direction arrows.

Groq Cloud

Llama 3.3 70B Versatile for AI navigation, chatbot, admin assistant, and task extraction.

OSRM Routing

Road-snapped polylines for walking, driving, and cycling profiles with alternatives.

JSON File Storage

No database — flat JSON files with auto-increment IDs and LOCK_EX for write safety.

 IoT — ESP32-CAM System

Onboard bus camera portal using ESP32-CAM (OV3660) for live feed, recording, and automatic cloud upload.

Live Camera Feed

Web UI with live MJPEG stream, flash slider control, and debug console.

SD Card Recording

Auto-start on boot, 10-second segments, playback in UI, auto cleanup at 90% usage.

Captive Portal

AP name: bus#1, auto redirect to 1.2.3.4, Wi-Fi scan list for internet connection.

Auto Upload

1 image per minute to API endpoint. NTP time sync for Kathmandu timezone.
